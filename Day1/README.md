# Day 1

## Info - Qt Overview
Clone the TekTutor Qt Training Repository
```
git clone https://giqtthub.com/tektutor/qt-dec-2025.git
cd qt-dec-2025
```

Generate Qt project file
```
cd ~/qt-dec-2025
git pull
cd Day1/QtHello
ls -l
qmake -project
```

Generate Makefile
```
cd ~/qt-dec-2025
cd Day1/QtHello
ls -l
qmake
cat Makefile
```
